== Theme Description ==

Romana is a WordPress theme for startups and small and medium sized companies. Romana is a mobile responsive WordPress theme which looks impressive in Mobile and Tablets. Romana is customizable through customizer options. Logo, colors etc can be easily changed.

== Features ==

Responsive design(Bootstrap), Customization Options, Custom Menu, supports social media.

== Installation ==

1. Primary:
 = Login to your wp-admin account and go to Appearance -> Themes.
 = Select "Install" tab and click on the "Upload" link.
 = Select "romana.zip" and click on "Install Now" button.
 = In case of errors, use an alternate method.
 
2. Alternate:
 = Unzip the template file (romana.zip) that you have downloaded.
 = Upload the entire folder (romana) to your server via FTP and place it in the /wp-content/themes/ folder.
 = Do not change the directory name.
 = The template files should be there now: /wp-content/themes/romana/index.php (example).
 
3. Log into your WP admin panel and click on "Appearance". Go to "Themes" tab.

4. Now click on "Romana" to activate the theme.

5. Complete all of the required inputs in the Appearance => Custimize option and click "Save & Publish" to make the changes LIVE.
