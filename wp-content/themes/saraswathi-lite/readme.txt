== Description ==
Saraswathi Lite is a super light WordPress theme developed with WordPress's core philisohy in mind of "Clean, Lean and Mean'. It integrates with WordPress's native Customizer optons panel so you can change themes looks without coding. Whether you're a normal user, WordPress developer, or both - you will enjoy using this theme with a extensively commented codebase and clean design.

* Mobile-first, Responsive Layout
* Google Fonts
* Custom Background Colour and Image
* Custom Logo
* Grid or Full Width Layout
* Option to change the grid size
* Option to change content and sidebar size
* 4 blog layouts
* 60 inbuilt SVG Icon Fonts
* Left, Right or No Sidebar
* 3 page templates
* Customizer Theme Options
* 1 Sidebar, 4 Footer Widget Areas & Custom Frontpage Widget Areas
* Custom Widget Options
* Social Links
* Fully built with Native Customizer API
* Clean, Lean and Mean.
* Standard, Image and Video Post Formats
* Featured Image ( All Thumbnail, Medium, Large or Full )
* Enable/Disable Each Meta Information On Posts
* Enable/Disable Author Bio
* Umlimited colours for buttons, icons and forms.
* Footer Secondary Menu
* Scroll To Top
* Page Preloader
* Waypoint CSS3 Reveal Animation
* The GPL v2.0 or later license. :) Use it to make something cool.

For more information about Saraswathi Lite please go to http://www.smartpixelsdemo.com/saraswathi-lite/.

== Installation ==

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Type in Saraswathi Lite in the search form and press the 'Enter' key on your keyboard.
3. Click on the 'Activate' button to use your new theme right away.
4. Navigate to Appearance > Customize in your admin panel and customize to taste.
