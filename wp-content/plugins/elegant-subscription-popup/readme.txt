== Description ==
Elegant Subscription Popup is the <strong>most popular lead capturing wordpress plugin (7000+ downloads)</strong> that helps to convert your visitors to subscribers, this plugin supports Feedburner, Mailpoet and many other newsletters.

Elegant subscription popup plugin shows an animated <a href="http://techsini.com/our-wordpress-plugins/elegant-subscription-popup" target="_blank">feedburner form</a> when site gets loaded. It doesn't annoy the visitor to subscribe repeatedly instead it shows the feedburner subscription form with the interval of days set by admin.

Elegant subscription popup is responsive therefore it adjusts itself with PC, tablet and mobile devices.

<strong> Here are some cool features of Elegant Subscription Popup plugin</strong>

*   Cool animation effect on load and exit
*	Doesn't annoy your blog reader
*   Add your own logo image
*   Clean and elegant look
*   Responsive design
*   Easy to use admin panel

== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.
